# SESA slides 

--

## Usage

*Create a slide deck*
```
touch _slides/new-slideshow.md
$EDITOR _slides/new-slideshow.md
```

## Markdown in reveal.js

Example:

``` text
# Title

* Point 1
* Point 2

> This is a quote 

Note: this is a hidden speaker note

---

## Third Slide

==

## Vertical Slide 1

==

## Vertical Slide 2

```
